# 本地后台使用说明

## 启动

在项目目录运行：

```bash
npm run dev:all
```

打开：

- 作品集预览：http://127.0.0.1:5173
- 本地后台：http://127.0.0.1:8787/admin

## 后台能做什么

- 选择网页中的图片/视频位置
- 上传图片或视频到腾讯云 COS
- 自动写回 `public/content/portfolio-content.json`
- 前台作品集自动读取新素材
- 粘贴外部图片/视频链接并保存到指定位置
- 在作品集页面右下角继续使用“编辑文字”，保存后会写入后台配置

## COS 配置

后台优先读取 `.env.local`，如果没有，会兼容读取旧工具：

```text
tools/cos-config.local.js
```

`.env.local` 不要上传 GitHub。

## 发布逻辑

访客访问 GitHub Pages 时，不需要你的本地后台在线。

本地后台只用于编辑、上传素材和生成配置；发布后的网页会读取已经写入项目的内容配置，并加载腾讯云 COS 上的图片/视频。
